# 🌐 Web Development Internship Projects

This repository contains my completed tasks for the **Codveda Web Development Internship Program**. Each level focuses on different skills, progressing from basic to advanced web development concepts.

---

## 📁 Project Structure

```
├── level-1
│   ├── task-1 (Static Website)
│   ├── task-2 (Portfolio Webpage)
│   └── task-3 (JavaScript Interactivity)
│
├── level-2
│   ├── task-1 (Responsive Design)
│   ├── task-2 (To-Do List App)
│   └── task-3 (React/Vue App)
│
├── level-3
│   ├── task-1 (Full-Stack CRUD App)
│   └── task-2 (Authentication System)
```

---

## 🚀 Level 1 – Basic

### 🔹 Task 1: Static Website

- Built using HTML and CSS
- Includes homepage, about, and contact pages
- Responsive design using media queries

### 🔹 Task 2: Portfolio Webpage

- Single-page portfolio layout
- Sections: About, Projects, Contact
- Custom styling with animations and fonts

### 🔹 Task 3: JavaScript Interactivity

- Interactive UI elements (dropdowns, modals)
- Form validation using JavaScript
- Event handling implementation

---

## ⚡ Level 2 – Intermediate

### 🔹 Task 1: Responsive Web Design

- Layout using Flexbox and CSS Grid
- Fully responsive across devices

### 🔹 Task 2: To-Do List App

- Add, delete, and complete tasks
- Data stored using Local Storage

### 🔹 Task 3: Front-End Framework (React/Vue)

- Component-based architecture
- Basic state and props usage

---

## 🧠 Level 3 – Advanced

### 🔹 Task 1: Full-Stack CRUD Application

- Backend with Node.js / Express or Django
- REST API implementation
- Database integration (MongoDB/PostgreSQL)

### 🔹 Task 2: Authentication System

- User login and registration
- JWT or session-based authentication
- Secure routes and password hashing

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- React / Vue
- Node.js / Express
- MongoDB / PostgreSQL
